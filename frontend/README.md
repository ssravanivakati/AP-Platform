# AI Accounts Payable Platform

## Project Overview

AI Accounts Payable Platform is a frontend dashboard application designed to manage invoices, vendors, payments, and notifications in an accounts payable workflow.

## Technologies Used

- React.js
- TypeScript
- Tailwind CSS
- React Router
- Recharts
- Lucide React
- Vite

## Features

### Dashboard
- Invoice summary cards
- Invoice status overview
- Payment analytics charts
- Recent invoice table

### Invoice Management
- Invoice upload interface
- Invoice listing
- Invoice details view

### Vendor Management
- Vendor list
- Vendor dashboard
- Vendor performance tracking

### Notifications
- Invoice approval alerts
- Payment notifications
- Read/unread status

### Settings
- User profile section
- Company settings
- Notification preferences
